03

놓치면 안 되는 의존성 경고

반드시 함께 가거나, 초기에 착수해야 하는 것들
  • 자유 발화 모드 세트 — 자유 발화 UI + 청크 업로드 + Whisper 연동 + 분배 엔진 + 검증 도구. 이 중 하나라도 빠지면 전체 기능 불가.
  • 카톡 알림톡 조기 신청 — 템플릿 승인 1~3주 소요. 개발 시작과 동시에 카카오 비즈메시지 신청 필수.
  • Phase 2 자산 보존 구조 — DB 스키마 설계 시점에 결정. 나중에 추가하면 과거 데이터 유실. 자유 발화 원문 + 변리사 리뷰 로그는 처음부터 append-only로 설계.
  • 데이터 격리 미들웨어 — 백엔드 API 개발 초기에 미들웨어로 박아야 함. 나중에 끼워넣으면 전체 코드 수정 필요.
  • LLM 호출 추상화 레이어 — MVP 시작 시 Sonnet 4.7 기반으로 설계하더라도 모델 교체 가능한 구조로. 나중에 nano/gpt-5로 옮길 때 전체 재작업 방지.
  • PII 암호화 — 제품 초기부터 적용. 나중에 추가하면 DB 마이그레이션 + 다운타임 발생.
